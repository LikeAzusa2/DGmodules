# DGmodules

Additional modules for **Draconic Evolution** on **Minecraft 1.20.1 / Forge**.

DGmodules is an unofficial addon mod that adds extra modules and related
mechanics for Draconic Evolution, including content ported from the newer
1.21.1 branch to the 1.20.1 environment.

## Requirements

- Minecraft `1.20.1`
- Forge `47.x`
- Draconic Evolution `3.1.2.621+`
- Brandon's Core `3.2.1.302+`
- Curios `5.4.7+1.20.1` (optional at runtime, supported by the project)

## Development

- Java `17`
- Gradle `8.1`

Common commands:

```powershell
.\gradlew.bat build
.\gradlew.bat runClient
```

## Notes

- This project is provided as-is.
- Compatibility with other mods is not guaranteed.
- Back up existing worlds before testing new builds.

## Issues

When reporting a problem, include relevant logs or crash reports when possible:

- `run/logs/latest.log`
- files under `run/crash-reports/`
